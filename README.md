# Warriors Lens — Julio C. Hernandez

Portfolio site for SSG Julio C. Hernandez (U.S. Army Combat Photographer & Public Affairs Specialist, 55th Public Affairs Company / Combat Camera).

Static HTML / CSS / JavaScript site — no build step, no backend required.

## Live Site
Hosted on GitHub Pages: **https://jch00047-hue.github.io/warriorslens/**

## Local Preview
```bash
python3 -m http.server 8000
# open http://localhost:8000
```

## Editing
- Site copy & section markup: `index.html`
- Theme & layout: `styles.css` (dark + gold accent palette)
- Interactivity (gallery filters, lightbox, contact form): `app.js`
- Assignment cards data: `assignments.json`
- DVIDS photos: `media/`
- Coursework downloads: `files/`

## Updating
Edit files locally, then:
```bash
git add .
git commit -m "your update"
git push
```
GitHub Pages re-deploys automatically within ~1 minute.
